# ArchBrain Neural Sentinel

An autonomous architectural auditing tool for Clean Architecture projects.

## Structure

- `/backend`: The Node.js analysis engine and MCP server.
- `/ui`: The modern React/TSX dashboard.
- `/data`: Persistent logs, neural pulses, and user preferences.
- `/scripts`: Internal debug and utility scripts.
- `/legacy`: Old static HTML versions of the tool.

## Commands (from Project Root)

- `npm run build-brain`: Rebuilds the React frontend.
- `npm run start-brain`: Starts the analysis server and serves the UI at :5050.
